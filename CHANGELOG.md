# DevSandbox Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-05-10

### Added
- Initial project foundation and monorepo structure
- React + Vite + TypeScript frontend application
- NestJS backend API with modular architecture
- Python FastAPI analysis service
- PostgreSQL database setup
- Redis cache and queue integration
- Docker and Docker Compose configuration
- Comprehensive documentation (README, ARCHITECTURE, SETUP, CONTRIBUTING, DEPLOYMENT)
- Health check endpoints in backend and analysis service
- Basic authentication module placeholder
- Analysis job management endpoints
- Scalable folder organization
- ESLint and Prettier configurations
- GitHub Actions CI/CD workflow template
